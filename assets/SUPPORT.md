# Support

## Public Support Scope

Public support for THE HOOK is limited to:

- public repository navigation
- public documentation questions
- public landing page issues
- broken public links
- clarification of the Public Edition boundary
- responsible disclosure routing

Public support does not include private implementation support, deployment guidance, private enterprise access, operational deception design, telemetry pipeline review, detection rule review, token generation methods, client environment review, or commercial implementation detail.

## Private Enterprise Discussions

Private enterprise access, licensing, acquisition, design-partner, or implementation discussions must be handled outside public GitHub channels.

Schedule a private discussion:

https://cal.com/ciprian-stefan-plesca

## Security Support

For security concerns, do not open a public issue.

Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This repository does not provide guaranteed public support response times.

Any paid support, private enterprise assistance, diligence support, implementation support, or transfer support must be defined in a separate written agreement.

## Public Communication Rules

Do not post any of the following in public issues, discussions, or pull requests:

- secrets, tokens, keys, or credentials
- operational deception logic
- telemetry pipelines
- detection rules
- token generation methods
- client environment details
- private playbooks
- prompts or AI orchestration details
- customer or buyer data
- private enterprise materials
- third-party confidential information
